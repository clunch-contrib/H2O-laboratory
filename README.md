# [H2O-laboratory](https://clunch-contrib.github.io/H2O-laboratory/) : 水分子式H2O实验用例，基于image3D开发，用于探索三维几何坐标运算库的最佳实践。

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/clunch-contrib/H2O-laboratory/issues)，欢迎参与维护！

## How to use?

开发的时候运行下列命令：

```
npm run dev
```

开发完毕需要发布的时候，请运行下列命令：

```
npm run build
```
